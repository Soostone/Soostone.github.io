                                          API Reference

                                          Napkin currently exposes all internal APIs. However, it is recommended to import just one modules from Napkin (to write custom haskell queries): Napkin.Backends.$BACKEND, where $BACKEND can be one of:

                                          • BigQuery
                                          • Postgres
                                          • Redshift
                                          • Sqlite
